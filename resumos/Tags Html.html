Principais Tags HTML

Header:

Tag utilizada para criar títulos. Pode variar em 6 tamanhos diferentes sendo 1 o maior e 6 o menor.

Uso em código: "< h1 > content < / h1 >"

= parágrafo

= citação = itálico ou = negrito (bold) = sublinhado (underline) = marcador de texto = coloca o texto acima menor = link = cria uma etiqueta (geralmente utilizado antes de inputs) _____________
= tag de formulário = cria campos para o usuário inserir algo text = texto number = números button = botão clicável a ser programado range = barra que alterna entre valores color = caixa de cor e-mail = email (exige @) url = link (exige http ) date = data week = semana (só funciona no edge e chrome) checkbox = seleção múltipla radio = seleção de apenas uma opção (atentar ao nome) file = arquivos (pode ter multiple) hidden = envia algo escondido na tela search = busca (só funciona no chrome) action = define para onde as informações serão enviadas
